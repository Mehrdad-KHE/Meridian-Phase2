# پیش‌تحویل

قبل از نهایی‌کردن و آینه‌کردن هر خروجی، این موارد باید پاس شوند:

1. کلید کلاینت باید صریح باشد.
2. کلاینت نباید با کلاینت دیگر قاطی شده باشد.
3. تعداد سطرها باید با تعداد سندهای نهایی بخواند.
4. اگر سندهای تکراری وجود داشته‌اند، نقشه‌ی کانونی‌سازی باید ثبت شده باشد.
5. تاریخ‌ها باید یکدست باشند.
6. ردیف‌های منفی باید قرمز مانده باشند.
7. فیلدهایی که روی منبع نیستند نباید حدسی پر شده باشند.
8. خروجی نهایی باید از نظر نام‌گذاری و مسیر برای تحویل آماده باشد.

اگر یکی از این موارد رد شد، خروجی هنوز نهایی نیست.
